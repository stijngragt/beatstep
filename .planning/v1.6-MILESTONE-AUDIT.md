### Integration Issue

**PlaylistListView "Select for Run" → RunTabView**

- **Location:** `BeatStep/Views/Library/PlaylistListView.swift` context menu
- **Problem:** Calls `selectedTab.wrappedValue = .run` without setting `LastRunPlaylist.id/name/imageURL`
- **Effect:** User taps "Select for Run" from library list, arrives at Run tab showing "no playlist" state
- **Workaround:** The path through PlaylistDetailView → "Run with this playlist" works correctly
- **Fix:** Add `LastRunPlaylist.id = playlist.id`, `.name = playlist.name`, `.imageURL = playlist.images?.first?.url` before the tab switch

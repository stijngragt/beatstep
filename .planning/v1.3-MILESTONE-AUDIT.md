## Unsatisfied Requirements

### PLR-04: Toggle 1:1/1:2 tempo matching mid-run

**Phase:** 13 (Engine Extensions)
**What exists:** `RunEngineService.tempoMode` property, `TempoMode` enum with UserDefaults persistence, `adjustedCadence` computation using cadence/2 in half mode, `findMatchingTracks` ranking preferring cadence/2 tracks.
**What's missing:** No view in the entire codebase reads or mutates `tempoMode`. Zero tap targets exist for the user to change tempo mode mid-run.
**Fix scope:** Small — add a toggle button to `ActiveRunView` or `RunPlayerView` that calls `runEngine.tempoMode = (runEngine.tempoMode == .oneToOne ? .half : .oneToOne)`.

**2. Add FilterChipRow inline component (LIB-02):**
Create a private struct `FilterChipRow` inside PlaylistListView.swift:
```swift
private struct FilterChipRow: View {
    @Binding var activeFilter: PlaylistFilter

    var body: some View {
        HStack(spacing: Spacing.sm) {
            ForEach(PlaylistFilter.allCases, id: \.self) { filter in
                Button {
                    BSHaptics.selection()
                    withAnimation(BSAnimation.snappy) {
                        activeFilter = filter
                    }
                } label: {
                    Text(filter.rawValue)
                        .font(.captionBold)
                        .padding(.horizontal, Spacing.md)
                        .padding(.vertical, Spacing.xs)
                        .background(
                            activeFilter == filter ? Color.accent : Color.surfaceOverlay,
                            in: Capsule()
                        )
                        .foregroundStyle(
                            activeFilter == filter ? Color.textOnAccent : Color.textSecondary
                        )
                }
                .buttonStyle(.plain)
            }
            Spacer()
        }
    }
}
```

Place filter chips as the FIRST item inside the List (before the scan progress banner) with `.listRowSeparator(.hidden)` and `.listRowInsets(EdgeInsets(top: Spacing.xs, leading: Spacing.md, bottom: Spacing.xs, trailing: Spacing.md))`. This makes them scroll naturally with content per user decision.

**3. Switch ForEach to filteredPlaylists (LIB-01 + LIB-02):**
- Change `ForEach(playlists)` to `ForEach(filteredPlaylists)`
- KEEP pagination trigger checking against `playlists.last?.id` (unfiltered) to avoid pagination mismatch -- per research pitfall #4

**4. Redesign PlaylistRow (LIB-03):**
Rewrite the `PlaylistRow` struct:
- Height: 70pt (up from 50pt, per user decision)
- Cover art: `ComponentSize.coverArtMedium` (56pt) with `Radius.sm` rounded corners
- Keep existing AsyncImage pattern for cover art
- VStack with: playlist name (bodyText, semibold, lineLimit 1), track count (captionText, textSecondary), coverage bar OR scanning progress OR "Not analyzed" text
- Coverage bar: Use inline `CoverageBar` private struct:
```swift
private struct CoverageBar: View {
    let coverage: PlaylistCoverage

    var body: some View {
        HStack(spacing: Spacing.sm) {
            GeometryReader { geometry in
                ZStack(alignment: .leading) {
                    RoundedRectangle(cornerRadius: 2)
                        .fill(Color.surfaceOverlay)
                    RoundedRectangle(cornerRadius: 2)
                        .fill(coverage.statusColor)
                        .frame(width: geometry.size.width * coverage.percentage)
                }
            }
            .frame(height: 4)

            Text(coverage.text)
                .font(.captionText)
                .foregroundStyle(Color.textSecondary)
        }
    }
}
```
- When scanning: show inline ProgressView + "Analyzing X/Y" (same as current pattern)
- When no coverage and coverageLoaded: show "Not analyzed" in captionText, textTertiary (per user decision -- subtle)
- Text format: "42/50 BPM" alongside the bar (per user decision)

**6. Add context menu (LIB-04):**
Apply `.contextMenu` on each NavigationLink row:
```swift
.contextMenu {
    if coverage != nil {
        Button {
            BSHaptics.medium()
            Task {
                await scanService.scanPlaylistByID(playlist.id, name: playlist.name)
                loadCoverageData()
            }
        } label: {
            Label("Re-scan", systemImage: "arrow.clockwise")
        }
        Button(role: .destructive) {
            BSHaptics.warning()
            scanService.deleteScan(playlistID: playlist.id)
            loadCoverageData()
        } label: {
            Label("Delete Scan", systemImage: "trash")
        }
    } else {
        Button {
            BSHaptics.medium()
            Task {
                await scanService.scanPlaylistByID(playlist.id, name: playlist.name)
                loadCoverageData()
            }
        } label: {
            Label("Analyze BPM", systemImage: "waveform.badge.magnifyingglass")
        }
    }
    Button {
        selectedTab.wrappedValue = .run
    } label: {
        Label("Select for Run", systemImage: "figure.run")
    }
}
```

Note: For "Select for Run", add `@Environment(\.selectedTab) private var selectedTab` to PlaylistListView. This follows the established SelectedTabKey pattern from Phase 25.

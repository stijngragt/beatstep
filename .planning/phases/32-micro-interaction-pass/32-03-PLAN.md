    **CRITICAL for run screen (D-08):** Animation drivers on ActiveRunView conditionals MUST be scoped to specific state values. Do NOT add a blanket `.animation()` to the ActiveRunView body. Scope each `.animation()` to the specific value that drives that conditional.

    **ActiveRunView.swift — 4 conditional views:**
    1. `if runEngine.runMode == .guided, let phase = ...` (RampPhaseIndicator): Add `.transition(.opacity)` to the RampPhaseIndicator view. Add `.animation(BSAnimation.smooth, value: runEngine.runMode)` scoped to just this conditional or its container.
    2. `if runEngine.runMode == .guided` (ZoneBandView): Add `.transition(.opacity)` to the ZoneBandView. The animation driver can be shared with #1 since both key on runMode.
    3. `if let track = ...` (RunPlayerView): Add `.transition(.opacity)` to the RunPlayerView. Add `.animation(BSAnimation.smooth, value: runEngine.currentMatchedTrack != nil)` scoped to this view's container (use a computed bool or the Optional itself).
    4. `if runEngine.runMode == .guided && rampPhase != .coolDown` (Cool Down button): Add `.transition(.opacity)`. Animation driver shared with the runMode-based animation.

    **RunTabView.swift — 2 conditional areas:**
    1. The main `if/else` state switching (noPlaylist/loading/loaded/error): Add `.transition(.opacity)` to each branch. Add `.animation(BSAnimation.smooth, value: stateVariable)` on the enclosing Group/container, keyed to whatever state drives the switching.
    2. `if let range = RunZone.mergedBPMRange(...)`: Add `.transition(.opacity)` to the range display view. Add `.animation(BSAnimation.snappy, value: selectedZoneIds)` if not already present (check — RunTabView may already have this).

    **RunStatusBar.swift — 1 conditional:**
    1. `if let zoneName`: Add `.transition(.opacity)` to the zone name text. Add `.animation(BSAnimation.smooth, value: zoneName)` on the parent.

    **CadenceDisplayView.swift — 1 conditional (careful with D-08):**
    1. `if isGuidedMode` / else (delta vs sync label): Add `.transition(.opacity)` to each branch. Add `.animation(BSAnimation.smooth, value: isGuidedMode)` on parent. This is safe because isGuidedMode only changes once per run (when switching modes), not rapidly. The number text inside these branches must NOT have animation modifiers.

    **RunPlayerView.swift — 1 conditional:**
    1. `if let bpm = trackBPM`: Add `.transition(.opacity)` to the BPM display. Add `.animation(BSAnimation.smooth, value: trackBPM)` on the parent container.

    **MiniPlayerView.swift — 2 conditionals:**
    1. `if let track = playerService.currentTrack` (entire view content): Add `.transition(.opacity)`. Add `.animation(BSAnimation.smooth, value: playerService.currentTrack != nil)`.
    2. `if let bpm = currentBPM` / else: Add `.transition(.opacity)` to each branch. Add `.animation(BSAnimation.smooth, value: currentBPM)` on parent.

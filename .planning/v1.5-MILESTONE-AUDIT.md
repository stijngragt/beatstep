## Notable Integration Detail

`SelectedTabKey` (Phase 25) is declared as a `private struct` inside `ContentView.swift`. The `EnvironmentValues` extension exposing the `selectedTab` computed property is accessible to `PlaylistDetailView` because the extension's computed property has default (internal) access. This works correctly but is a maintenance risk if the key is ever moved to a separate file.

    - selectedZoneId returns the stored zone ID (1-5) when set
    - Setting selectedZoneId to nil stores 0 (Free)
    - Setting selectedZoneId to a zone ID stores that value
    - When selectedZoneId is non-nil and matches a zone, runMode should be .guided and targetBPM should be zone.bpm
    - When selectedZoneId is nil, runMode should be .free
  </behavior>
  <action>

   - Private key "selectedRunZoneId"
   - `get`: read UserDefaults integer; return nil if 0, otherwise the Int
   - `set`: write newValue ?? 0 to UserDefaults

2. Create `BeatStep/Views/Run/ZonePickerView.swift`:
   - `@Binding var selectedZoneId: Int?` (nil = Free)
   - Private computed `zones: [RunZone]` reading `RunZone.saved`
   - Body: `ScrollView(.horizontal, showsIndicators: false)` containing `HStack(spacing: Spacing.sm)`:
     - `ForEach(zones)` rendering zone capsule buttons
     - One "Free" capsule button at the end
   - Zone capsule: two-line VStack with `zone.displayLabel` (Font.captionBold) and `"\(zone.bpm)"` (Font.labelText, Color.textSecondary). Padding .horizontal Spacing.md, .vertical Spacing.sm. Background Capsule fill: surfaceOverlay when selected, surfaceElevated when not. ForegroundStyle: textPrimary selected, textSecondary not.
   - Free capsule: single-line "Free" (Font.captionBold). Same padding and styling. Use `.frame(minHeight:)` matching the two-line capsule height to prevent vertical misalignment (research pitfall 3).
   - On zone tap: set selectedZoneId = zone.id
   - On Free tap: set selectedZoneId = nil
   - HStack padding .horizontal Spacing.xs

   - testFreeModeMappingWhenNil: selectedZoneId nil means free mode
